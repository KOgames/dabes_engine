collection:add (member, ...)
Add a member to the Collection. This caches the member so it doesn't get garbage collected prematurely. It also calls the adder hook with any variable arguments.

Parameters:

  • member The member to add to the collection
  • ... Arguments to be passed to adder

Returns:

    Whatever adder returns
collection:remove (member, ...)
Remove a member from the Collection. This removes the member from the cache so it can get garbage collected. It also calls the remove hook with any variable arguments.

Parameters:

  • member The member to remove the collection
  • ... Arguments to be passed to remover

Returns:

    Whatever remover returns
